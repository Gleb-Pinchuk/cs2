# Coursework 2

Приложение на Python для получения вакансий с платформы hh.ru через их API, сохранения в JSON-файл и удобной работы с данными: добавление, фильтрация, сортировка и удаление вакансий.

---

## Возможности

- Получение вакансий по запросу через API hh.ru  
- Сохранение вакансий в JSON-файл  
- Фильтрация вакансий по:
  - ключевым словам  
  - диапазону зарплаты  
- Сортировка по уровню зарплаты  
- Удаление вакансий из файла  
- Получение топ-N вакансий по зарплате  
- Тесты для проверки базового функционала
